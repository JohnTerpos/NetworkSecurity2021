# NetworkSecurity2021 Project DDOS ATTACK 

## 1) Περιγραφή συστήματος
Το σενάριο της επίθεσης DDOS θα υλοποιηθεί με την αποστολή υπερβολικά μεγάλου αριθμού ψεύτικων αιτήσεων για εξυπηρέτηση με αποτέλεσμα η υπηρεσία να μη μπορεί να εξυπηρετήσει αυτούς που πραγματικά θέλουν την υπηρεσία.
